# ESP-NOW 无线测试固件

## 项目简介
ESP-NOW 是 Espressif 开发的低功耗无线通信协议，基于 Wi-Fi 但无需连接路由器。本套固件用于测试 ESP32/ESP8266 设备间的点对点无线通信功能。

## 技术特点
- **传输速率：** 最高 250kbps
- **通信距离：** 室内 30-50 米，室外 100-200 米
- **功耗：** 极低功耗，适合电池供电设备
- **延迟：** 毫秒级传输延迟
- **拓扑：** 支持一对一、一对多、多对一通信

## 硬件要求
1. ESP32 或 ESP8266 开发板
2. USB 数据线
3. 电脑已安装 CP210x 或 CH340 驱动
4. ESP32 Flash Download Tools

## 配对说明
ESP-NOW 通信需要先进行设备配对：
1. 发送端和接收端需要烧录对应的固件
2. 首次上电时会自动扫描附近的设备
3. 通过串口可以查看配对状态
4. 配对成功后即可开始数据传输

## 测试步骤

### 基础通信测试
1. 准备两块开发板，分别烧录发送端和接收端固件
2. 上电后观察串口输出，确认设备初始化成功
3. 等待设备自动配对（通常需要 10-30 秒）
4. 在发送端按下按键，观察接收端是否收到数据
5. 检查数据完整性和传输延迟

### 距离和信号测试
1. 将发送端和接收端放置在不同距离
2. 记录不同距离下的信号强度（RSSI）
3. 测试极限传输距离
4. 观察障碍物对信号的影响

### 多设备测试
1. 使用多个发送端设备
2. 测试接收端同时接收多路数据的能力
3. 验证数据冲突处理机制

## 常见问题

### 配对失败
- **原因：** 设备距离太远或存在干扰
- **解决：** 缩短设备距离，避免 2.4GHz 干扰源

### 数据丢失
- **原因：** 信号弱或信道冲突
- **解决：** 调整设备位置，更换 WiFi 信道

### 无法烧录
- **原因：** 驱动问题或进入下载模式失败
- **解决：** 重新安装驱动，按住 BOOT 键再连接 USB

## 技术参数
- **工作频段：** 2.4GHz ISM 频段
- **调制方式：** GFSK
- **数据速率：** 1Mbps（物理层）
- **加密方式：** CCMP 加密
- **最大设备数：** 理论上支持 20 个配对设备
